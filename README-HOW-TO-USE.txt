PulsePair root-folder website

Files included in root:
- index.html
- product.html
- cart.html
- checkout.html
- success.html
- contact.html
- faq.html
- style.css
- app.js
- all product images

How to use:
1. Upload all files to a new GitHub repo root.
2. Connect repo to Cloudflare Pages.
3. Build command: leave empty
4. Output directory: /
5. Deploy.

This version is static and frontend-only.
Checkout and success work locally with localStorage.
Real payment gateway is not connected yet.

To change product images later:
- Replace the image files in the root folder using the same names.
